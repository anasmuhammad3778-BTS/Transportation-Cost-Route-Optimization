🚚 Transportation Cost & Route Optimization Dashboard
📌 Project Overview

This project presents an interactive Power BI dashboard designed to analyze transportation operations and logistics performance. It enables supply chain managers to monitor freight cost, fuel expenses, delivery efficiency, shipment trends, and route performance through data-driven insights.

Business Problem

Transportation contributes significantly to overall logistics cost. Organizations need visibility into:

Freight Cost
Fuel Consumption
Route Efficiency
Delivery Performance
Customer-wise Transportation Spend
Shipment Priority
Delay Analysis

This dashboard helps identify operational inefficiencies and supports informed decision-making.

Business Insights
Transportation spend exceeds ₹38 Million.
Long-haul routes contribute the highest freight cost.
Diesel vehicles generate maximum fuel expense.
Shipment delays account for nearly 31% of deliveries.
Customer-wise freight spending identifies high-value logistics accounts.
