# 🚚 Transportation Cost & Route Optimization Dashboard

> An interactive Supply Chain Analytics dashboard built using **Power BI, DAX, Power Query, and Excel** to analyze transportation cost, route efficiency, shipment performance, and logistics operations.

---

## 📖 Project Overview

Transportation is one of the largest contributors to logistics cost in any supply chain. This project provides an executive-level dashboard that helps monitor transportation performance by analyzing freight cost, fuel expenses, shipment delays, customer-wise transportation spend, and route efficiency.

The dashboard is designed to support data-driven decision-making for logistics and supply chain managers through interactive KPIs, dynamic filters, and insightful visualizations.

---

## 🎯 Business Objectives

This dashboard aims to answer key business questions such as:

- Which customers generate the highest transportation cost?
- Which routes are the most expensive?
- What is the overall delivery success rate?
- How much is being spent on fuel?
- Which shipment priorities dominate operations?
- How does freight cost vary over time?
- How do route distance and freight cost correlate?
